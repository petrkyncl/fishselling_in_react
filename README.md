# Fin & Gill

React + Tailwind recreation of the seafood storefront.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy To Vercel

This repo is configured for Vercel with `vercel.json`.

1. Import the repository in Vercel.
2. Use the default build command: `npm run build`.
3. Deploy the `dist` output directory.
